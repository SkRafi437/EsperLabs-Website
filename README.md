## How to Run Locally

```bash
git clone https://github.com/YOURUSERNAME/esperlabs.git
cd esperlabs
# Open index.html in your browser
open index.html
```

---

## How to Deploy (GitHub Pages)

1. Push `index.html` to the `main` branch
2. Go to **Settings → Pages**
3. Set source to `main` branch → `/ (root)`
4. Save — site goes live at `https://YOURUSERNAME.github.io/esperlabs`

---

## Connecting a Custom Domain Later

When you're ready to use a domain like `esperlabs.in`:

1. Buy the domain (Hostinger, GoDaddy, etc.)
2. Add a file named `CNAME` to your repo with just this inside:
   ```
   esperlabs.in
   ```
3. In your domain provider's DNS settings, add these records:
   ```
   A     @     185.199.108.153
   A     @     185.199.109.153
   A     @     185.199.110.153
   A     @     185.199.111.153
   CNAME www   YOURUSERNAME.github.io
   ```
4. Back in GitHub → Settings → Pages → add your custom domain
5. Check **Enforce HTTPS**

Done — your site will be live at `esperlabs.in` within a few hours.
